# 🍳 Recipe Finder

A responsive and user-friendly web application designed to help users search and discover delicious recipes through a simple, clean, and interactive interface.

## 🌐 Live Demo

🚀 **[View Live Application](https://recipe-finder-main-umber.vercel.app/)**

---

## ✨ Features

- 🔍 Search and discover different recipes
- 🍽️ Explore recipe information easily
- 🎨 Clean and user-friendly interface
- 📱 Responsive design for different screen sizes
- ⚡ Fast and simple user experience
- 🌐 Live application deployed using Vercel

---

## 🛠️ Technologies Used

- **HTML5** — Structure of the application
- **CSS3** — Styling and responsive design
- **JavaScript** — Application functionality and interactivity
- **Vercel** — Application deployment

## 🚀 Getting Started

Follow these steps to run the project locally.

### 1. Clone the Repository

```bash
git clone https://github.com/lavanya-0005/Recipe-Finder.git
```

### 2. Open the Project Folder

```bash
cd Recipe-Finder
```

### 3. Run the Application

Open the `index.html` file in your web browser.
